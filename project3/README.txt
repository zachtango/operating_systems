# Project 3 Zach Tang CS 4348.006

## Compiling and Running the Project
1. To compile, run `make`
- this will produce an executable named `file`

2. To run, execute `./file`
- this will give an option to choose the file allocation (1 - 3)
- once the file allocation is chosen, the user can interact with the file system
    through the provided interface
